# util.uuid
**Tap the page to copy a fresh UUID.**

This is a tool I made for myself. I was updating a test file that required me to generate over 150 random UUIDs and manually copy them over one by one. 

I would typically reach for the Node/DevTools REPL and type out crypto.randomUUID() and copy the results. It works for small stuff, but I wanted REPL speed with a slightly better experience. 

Feel free to use it if it saves you a few precious seconds.
